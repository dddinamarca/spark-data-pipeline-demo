# Spark Data Pipeline Demo

End-to-end data engineering pipeline demonstrating real-time data ingestion and processing using Apache Spark, Kafka and Delta Lake.

## Architecture

Data Generator → Kafka → Spark Structured Streaming → Delta Lake → Analytics

## Technologies

- Apache Spark
- Kafka
- Delta Lake
- Python
- Docker

## Features

- Real-time event ingestion
- Stream processing
- ETL transformations
- Data Lake storage
- Analytical queries

## Use Case

Simulated sales event processing pipeline similar to real-world e-commerce data pipelines.
